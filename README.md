# Pokemon React JS

This is a React application that showcases Pok√©mon-related data and interactions. The project is bootstrapped with Create React App, making it straightforward to run and customize.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode at  [http://localhost:3000](http://localhost:3000).

The page reloads on changes, and lint errors appear in the console.

### `npm test`

Launches the test runner in the interactive watch mode.

### `npm run build`

Builds the app for production in the `build` folder, minifying files for optimized performance.

### `npm run eject`

Exposes configuration files for advanced customization (not reversible).
